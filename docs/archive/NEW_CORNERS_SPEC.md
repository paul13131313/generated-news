# 新コーナー仕様書（2025/02）

generated-newsに追加する5つの新コーナーの企画・実装仕様。

## 購読者プロファイル拡張

現行の「関心カテゴリ」「キーワード」に加え、以下の設定項目を追加する。

| 項目 | 入力形式 | 用途 | 例 |
|------|---------|------|-----|
| 居住エリア | テキスト or セレクト | 地域ニュース、天気、イベント情報 | 恵比寿、渋谷区 |
| 最寄り駅 | テキスト | 地域ニュースの精度向上 | 恵比寿駅 |
| 性別（任意） | セレクト | ファッション提案の参考 | 男性 / 女性 / 指定しない |
| 年代（任意） | セレクト | ファッション・イベントの傾向調整 | 30代 |

`index.html`の関心プロファイルモーダルにフィールドを追加し、Workers側でプロンプトに注入する。

## コーナー1: 地域ニュース

### 概要
購読者の居住エリアに特化したローカルニュース。初期実装は恵比寿特化。

### コーナー名案
「街の声」「まちの記憶」「ご近所新聞」など新聞らしい名前を検討

### 掲載内容
- 新店オープン・閉店情報
- 地域イベント（祭り、マルシェ、ワークショップ等）
- 再開発・工事情報
- 地元の話題（行列店、季節の風景など）

### データソース案
- Web検索API（Google Custom Search等）でエリア名＋ニュースを取得
- プロンプトで「{エリア名}の最新ローカル情報」を指示し、AIが構成
- 将来的にはGoogle Places API等で新店情報を自動取得も検討

### プロンプト設計方針
```
あなたは{エリア名}の地域情報に詳しい新聞記者です。
以下の情報源をもとに、{エリア名}の住民が関心を持つローカルニュースを
3〜4本、簡潔な記事形式で書いてください。
- 新店・閉店、イベント、再開発、季節の話題を優先
- 各記事は見出し＋本文2〜3文
- 住民目線の温かみのあるトーンで
```

### 表示イメージ
```

├─────────────────────────┤
│ ▸ 恵比寿ガーデンプレイス  │

│   プン。焙煎所併設で…    │
│                         │
│ ▸ 駅前広場で週末マルシェ  │
│   地元農家の野菜が並ぶ…  │
```

## コーナー2: 天気×ファッション

### 概要
購読者の居住エリアの天気予報＋気温に連動したファッション提案。

### コーナー名案
「今日の空と装い」「きょうの着こなし」など

### 掲載内容
- 今日の天気・気温・降水確率（午前/午後）
- 天気に合わせた服装提案（1コーデ）
- ワンポイントアドバイス（折りたたみ傘、日焼け止め等）

### データソース案
- **天気API**: OpenWeatherMap（無料枠あり）or WeatherAPI
  - エリア名 or 緯度経度で取得
  - 必要データ: 天気、最高/最低気温、降水確率、風速
- 取得した天気データをプロンプトに注入し、AIがファッション提案を生成

### Cloudflare Workers での実装方針
```javascript
// 1. 天気APIからデータ取得
const weatherRes = await fetch(
  `https://api.openweathermap.org/data/2.5/weather?q=Ebisu,Tokyo,JP&appid=${API_KEY}&units=metric&lang=ja`
);
const weather = await weatherRes.json();

// 2. 天気データをプロンプトに注入
const prompt = `
今日の${area}の天気: ${weather.weather[0].description}
気温: ${weather.main.temp}℃（最高${weather.main.temp_max}℃ / 最低${weather.main.temp_min}℃）
降水確率: ${rain}%
この天気に合わせたファッションを提案してください。
`;
```

### 表示イメージ
```
┌─────────────────────────┐
│ 🌤 今日の空と装い        │
│ 恵比寿 — 晴れ時々くもり  │
│ 18℃ / 9℃  降水確率 10%  │
├─────────────────────────┤
│ 薄手のニット＋ライトアウ │
│ ターがちょうどいい一日。  │
│ 朝晩は冷えるのでストール │
│ があると安心です。        │
│                         │
│ 💡 花粉シーズン到来。    │
│ マスクをお忘れなく。     │
└─────────────────────────┘
```

## コーナー3: 展示会・映画情報

### 概要
東京都内の展示会・映画公開情報。購読者の関心カテゴリとも連動。

### コーナー名案
「文化の窓」「きょうの一幕」など

### 掲載内容
- 開催中・近日開催の展示会（美術館、ギャラリー）
- 公開中・近日公開の映画
- 購読者の関心に合わせたピックアップ（1〜2本を詳しく紹介）
- 簡単なおすすめコメント

### データソース案
- Web検索で「東京 展示会 今週」「映画 公開 今週」等を取得
- 将来的にはイベントAPI（Peatix API等）や映画API（TMDb）の利用も検討
- 購読者の関心カテゴリ（アート、テクノロジー等）でフィルタリング

### プロンプト設計方針
```
以下の検索結果をもとに、今週東京都内で楽しめる展示会・映画情報を
まとめてください。
- 展示会2〜3件、映画1〜2件
- 各項目: タイトル、場所、期間、一言紹介
- 購読者の関心: {カテゴリ}
- 関心に合うものがあれば「おすすめ」として強調
```

## コーナー4: SNS・YouTubeトレンド

### 概要
その日のSNS・YouTubeで話題のコンテンツをダイジェスト。

### コーナー名案
「ネットの潮目」「今日のバズ」「デジタル巷談」など

### 掲載内容
- X（Twitter）のトレンドトピック 3〜5件
- YouTube急上昇動画 2〜3件
- なぜ話題なのかの簡潔な解説
- 購読者の関心に関連するものを優先

### データソース案
- Web検索で「X トレンド 今日」「YouTube 急上昇 日本」等を取得
- Google Trends APIの活用も検討
- X API（有料）やYouTube Data API（無料枠あり）で直接取得も将来検討

### プロンプト設計方針
```
今日のSNS・YouTubeのトレンドを、新聞のコラム風にまとめてください。
- X（Twitter）のトレンド3〜5件、YouTube急上昇2〜3件
- 各トピック: 見出し＋なぜ話題かの解説1〜2文
- ネット文化に詳しい記者のトーンで
- 購読者の関心: {カテゴリ} に関連するものがあれば優先
```

### 概要

### コーナー名案
「ひだまり」「よりみち」「小さな幸福」など

### 掲載内容

- 善意・親切の話

## 技術メモ

### 天気API（OpenWeatherMap）
- 無料枠: 1,000 calls/day
- エンドポイント: `api.openweathermap.org/data/2.5/weather`
- APIキーはCloudflare Workersの環境変数に格納

### Workers側の構成変更
- 各コーナーを独立した関数/モジュールとして実装
- 並列実行でレスポンス時間を短縮（Promise.all）
- エラー時は該当コーナーをスキップ（他コーナーに影響させない）
